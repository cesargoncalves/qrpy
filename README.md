# qrpy - opencv

## Download the source

```bash
git clone https://github.com/cesargoncalves/qrpy.git 
```

## Setup dependencies

```bash
./run.sh --setup_dependencies
```

## Encode data

```bash
./run.sh --write
```

write data to encoded qr code and save it in output.png

## Decode data

```bash
./run.sh --read
```

decode data from qrcode on input.png
